# Grape CLI

Grape is a CLI tool for managing your infrastructure, migrated from the legacy Python IDP installer.

## Features

- **Deployment Management**: Deploy infrastructure using Terraform and GitOps (ArgoCD).
- **Configuration**: Manage project configurations stored in Supabase.
- **State Management**: Automatic handling of infrastructure facts and state.
- **Authentication**: Secure device-based authentication flow.

## Getting Started

To install the CLI:

```bash
go install github.com/bobikenobi12/bb-thesis-2026/apps/cli
```

Or run from source:

```bash
cd apps/cli
go run main.go
```
