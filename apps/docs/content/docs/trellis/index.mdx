# Trellis Platform

Trellis is the central control plane for your infrastructure. It manages configurations, deployments, and users.

## Features

- **Configuration Management**: Create and version infrastructure configurations.
- **Cluster Management**: View connected clusters and their health status.
- **Deployment History**: Track who deployed what and when.
- **Team Collaboration**: Manage access and permissions.
